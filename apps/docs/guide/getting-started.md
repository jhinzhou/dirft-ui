# Getting Started

## Install

```bash
pnpm add @drift/ui
```

## Usage

```vue
<script setup lang="ts">
import "@drift/ui/styles.css";
import { Button } from "@drift/ui";
</script>

<template>
  <Button>Save</Button>
</template>
```
